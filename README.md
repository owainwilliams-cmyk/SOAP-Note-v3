WFR SOAP Note v3 iPhone App

This is an installable iPhone web app/PWA. It is the practical standalone iPhone version.

Install:
1. Upload this whole folder to an HTTPS host such as Netlify, Vercel, GitHub Pages, or your own web space.
2. Open the hosted index.html link in Safari on the iPhone.
3. Tap Share → Add to Home Screen.
4. Launch SOAP Note from the Home Screen.

It works offline after first load and stores notes locally on the phone.

PDF:
Tap PDF, then use the iPhone print/share screen to save or send as PDF. The front/back body diagrams and marker labels are included.

Privacy:
Notes are stored locally in browser storage. This app does not upload patient data anywhere.
